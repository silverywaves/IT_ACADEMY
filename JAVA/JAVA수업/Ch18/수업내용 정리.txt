GUI(Graphical User Interface)
	: 그래픽 기반의 사용자 인터페이스
	
	* CLI(Command Line Interface) :  키보드를 이용하여 텍스트 기반으로 프로그램 제어

<새로운 프레임으로 객체를 생성하여 프로그램 시작>
-> 레이아웃 객체를 사용해 기본 레이아웃을 정의한 후 화면구성에 필요한 컴포넌트(버튼, 스크롤바, 아이콘, 툴바 등)를 생성하여 레이아웃에 배치
-> 사용자와 상호작용(클릭, 키보드 입력, 마우스 이동)하는 이벤트 핸들러 등록
-> 발생한 이벤트 처리 코드 구현


JFrame : 메인 프레임
JPanel : 보조프레임(프레임에서 했던 레이아웃 적용과 각종 UI 배치 가능)
Component : JLabel 등 각종 UI 기능을 수행
